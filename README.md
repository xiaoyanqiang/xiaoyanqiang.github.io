## 植保作业APP更新内容
[XAircraft](http://www.xaircraft.com)

### V3.8.1 更新内容
# 地图
+ 优化一些内容显示。

# 返航
+ 手动返航可以选择返航点（需要植保测绘APP记录位置）。
+ 航线也可以设置返航点（需要植保测绘APP记录位置）。
+ 返航可以设置高度。
+ 手动返航如果线路穿过已知障碍物，会提示碰撞，但不会避障，请用户根据实际情况进行操作。

# 航线
+ 取消独立的辅助点设置，起始辅助点移到进入线设置里，返航辅助点移到返航线设置里。
+ 航线改版，一条航线分为“进入线”，“作业航线”，“返航线”三部分。
+ 进入线，表示飞机进入作业航线的蓝色段。
+ 返航线，表示飞机作业完成返航的橙色段。

# 往返航线
+ 航线执行过程，可在航线信息的航线参数里实时调整高度和速度。
+ 对范围选择进行改版，当线段超过30条，就会显示一个缩略图，全部可以拖拽和滑动。
+ 优化了往返航线的断点续飞功能，现在断点续飞也可以选段了。
+ 增加喷洒策略“平移喷洒”，打开后，飞机平移也会喷药，主要针对有斜边地形，平移线太长的情况。
+ 支持“进入线”、“返航线”的设置。

# 扫边航线
+ 对扫边航线范围选择进行改版，改版后更直观，速度更快。
+ 增加喷洒策略“单个喷头”，打开后，飞机喷洒只打开朝边界外的喷头（自动判断）。
+ 支持“进入线”、“返航线”的设置。

# 定点航线
+ 定点高度将“离树梢高度”和“植物最大高度”合并成一个设置项。
+ 取消定点悬停的植物半径设置项。
+ 暂时不支持“进入线”、“返航线”的设置。

# 自由航线
+ 支持“进入线”、“返航线”的设置。

# 定位
+ 取消了“搜索XRTK基站”，改为“搜索移动基站”和“搜索固定基站”两项。
+ 搜索移动基站，通过电台通道搜索用户架设的移动基站。
+ 搜索固定基站，通过网络获取基站差分数据（即云RTK）。

# 其它
+ 探照灯开关挪到点击无人机状态栏飞行速度小图标打开的界面里。
+ 小幅修改界面内容的显示。
+ 点击无人机状态栏的小图标（譬如卫星图标、药箱图标），可以看到一些相关的数据信息。
+ 动力系统的怠速测试现在支持单独选择。
+ 增加启动作业不起飞的异常报告。
+ 支持多国语言语音合成。

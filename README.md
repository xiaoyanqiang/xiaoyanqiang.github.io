## 植保作业APP更新内容
[XAircraft](http://www.xaircraft.com)

### V3.6.3 更新内容 2017/4/29
---

#### 无人机信息及操作
+ 增加返航确认对话框（因为手动返航是不避障的，确保安全）
+ 更改动力系统界面，显示详细的状态信息。
+ 修正锁定飞机依然能执行作业的问题。
+ 修正动力系统警告的电机编号不正确的问题。
+ 修正动力系统主板温度显示不正确的问题。
+ 增加超声波超量程警告提示。
+ 优化搜索基站算法，提高搜索速度，修正搜到基站ID为0的情况。

#### 航线任务
+ 优化航线避障检测，在安全距离小的情况下，提高避障成功率，但大家还是要注意航线安全的。
+ 优化航线生成算法，修正多个障碍物非常靠近时会产生航线生成失败的问题。
+ 修正扫边不显示预估药量的问题。
+ 修正断点续飞无法使用辅助点。
+ 修正取消上传航线后，无法再次滑动上传，需要再次创建航线才能上传。
+ 如果没有超声波模块，没有地形模块，仿地功能将不启用。
+ 辅助点设置按钮上会看到设置高度。
+ 辅助点的最低高度改为6米。
+ 仿地最低高度改为1.5米。
+ 螺旋航线在创建航线时显示总作业面积。

#### 作业任务
+ 修正添加地块时，需求地块过多导致无法正常加载的问题。
+ 修正有时候无法刷新地块数据的问题。
+ 优化作业记录上传规则，更及时上传。

#### 界面
+ 创建航线的时候，地图上不会显示当前正在执行的航线。
+ 修正作业完成，但已完成的往返线没有体现在选段界面上的问题。
+ 修正地图标尺有时候关不掉的问题。

#### 新功能
+ 增加找飞机功能，可在“无人机管理”中打开。提供分享飞机位置到飞防运营的功能，让大家一起帮你找飞机。

#### 其它
+ 修正一些文字显示内容。
+ 更新多语言内容。


### V3.6.1 更新内容 
---
#### 优化
+ 蓝色的起始线与橙色的返航线现在不止会对障碍物避障，还会沿边界内部避障了。但大家还是要注意安全。

#### 修正
+ 修正多处内容没有显示的问题。
+ 修正螺旋航线地块不显示完成百分比的问题。
+ 修正飞行任务状态里，显示螺旋航线进度的显示错误。
+ 修正重置A2后，导致闪退的问题。
+ 修正模块信息里P20 v2.5的无人机会提示P20 2017的定位模块的固件版本号。
+ 修正P20 v2.5由于固件版本限制导致无法作业的问题。
+ 修正雾化值显示错误的问题。
+ 修正喷洒量的显示。
+ 修正启用地图“跟随无人机”按钮后，无人机旋转时，地图没有旋转。

#### 更改
+ 辅助点的最低高度改为2米。

#### 增加
+ 增加两个用户提示，无人机飞行任务按钮、“仿地飞行”已放到飞行高度里设置。
+ 增加在规划螺旋航线时，已完成的螺旋点会在右上角显示一个绿色图标。
